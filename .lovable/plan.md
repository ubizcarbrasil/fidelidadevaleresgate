## Plano: Construtor de Páginas Unificado ✅

O Construtor de Páginas foi unificado em uma única rota `/page-builder` que combina:
- **Blocos estáticos** (V1): texto, botão, banner, ícone, divisor, espaçador
- **Blocos dinâmicos** (V2): sessões de ofertas, parceiros, cupons, links manuais

### Funcionalidades implementadas
1. ✅ Rota única `/page-builder` — V2 removido
2. ✅ Editor unificado com lista de blocos (estáticos + dinâmicos)
3. ✅ Drag-and-drop para reordenação de elementos estáticos
4. ✅ Preview em tempo real lado a lado (split view com painéis redimensionáveis)
5. ✅ Painel de propriedades contextual (estático vs dinâmico)
6. ✅ Configurações de página (título, subtítulo, busca, visibilidade)
7. ✅ Sub-editores para sessões dinâmicas e links manuais
8. ✅ Sidebar unificado com "Construtor de Páginas"
